# InkShift

Cards Given:

Spectrum, Motion, Do-Over, Cycle, Crisoy

## Description
 
InkShift is a 2D sidescroller puzzle game. You play as an ink blob whose objective is to reach a piece of paper at each level. 

Game Mechanics:

When your ink timer runs out, you will respawn as a different color ink blob, and each blob has a different ability. For example, red is heavy, yellow is light, green is sticky, and
blue is hollow. This contributes to the "Motion", "Cycle", and "Spectrum" cards.

Along with respawning, ghosts (NPCs) will do the actions that they previously did, again. This contributes to the "Do Over" card.

Play the game for "Crispy" sound effects.
